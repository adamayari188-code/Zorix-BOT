# بوت ديسكورد للتحكم بالخادم

هذا البوت مصمم ليمنحك تحكمًا كاملاً بخادم الديسكورد الخاص بك، بالإضافة إلى إمكانيات إرسال رسائل خاصة للأعضاء وحساب عددهم.

## الميزات

-   **حساب عدد الأعضاء:** يمكنك معرفة عدد الأعضاء في الخادم بسهولة.
-   **إرسال رسائل خاصة:** إرسال رسائل مباشرة إلى أي عضو في الخادم.
-   **تحكم كامل بالخادم:** (سيتم إضافة المزيد من الأوامر هنا لاحقًا لتغطية هذه الميزة).

## الإعداد والتشغيل

لإعداد وتشغيل البوت، اتبع الخطوات التالية:

### 1. المتطلبات

تأكد من تثبيت [Python 3.8+](https://www.python.org/downloads/) على جهازك.

### 2. تثبيت مكتبات بايثون

افتح سطر الأوامر (Terminal/Command Prompt) وانتقل إلى مجلد المشروع، ثم قم بتثبيت المكتبات المطلوبة:

```bash
pip install -r requirements.txt
```

### 3. إنشاء تطبيق بوت في ديسكورد

1.  انتقل إلى [بوابة مطوري الديسكورد](https://discord.com/developers/applications).
2.  سجل الدخول بحسابك في ديسكورد.
3.  اضغط على `New Application` (تطبيق جديد).
4.  أعطِ تطبيقك اسمًا، ثم اضغط على `Create` (إنشاء).
5.  في القائمة الجانبية اليسرى، اذهب إلى `Bot`.
6.  اضغط على `Add Bot` (إضافة بوت)، ثم `Yes, Do it!`.

### 4. الحصول على توكن البوت (Bot Token)

1.  في صفحة `Bot`، ضمن قسم `Build-A-Bot`، ستجد `Token`.
2.  اضغط على `Copy` لنسخ التوكن. **حافظ على سرية هذا التوكن! لا تشاركه مع أحد.**

### 5. تمكين الامتيازات (Privileged Gateway Intents)

هذه خطوة **مهمة جدًا** لكي يعمل البوت بشكل صحيح:

1.  في نفس صفحة `Bot`، قم بتفعيل الخيارات التالية ضمن `Privileged Gateway Intents`:
    -   `PRESENCE INTENT`
    -   `SERVER MEMBERS INTENT`
    -   `MESSAGE CONTENT INTENT`
2.  تأكد من حفظ التغييرات بعد تفعيلها.

### 6. دعوة البوت إلى خادمك

1.  في القائمة الجانبية اليسرى، اذهب إلى `OAuth2` ثم `URL Generator`.
2.  ضمن `SCOPES`، اختر `bot`.
3.  ضمن `BOT PERMISSIONS`، اختر الصلاحيات التي يحتاجها البوت. للتحكم الكامل، يمكنك اختيار `Administrator` (مسؤول)، أو اختر الصلاحيات المطلوبة لكل أمر:
    -   `Send Messages` (إرسال الرسائل)
    -   `Read Message History` (قراءة سجل الرسائل)
    -   `Manage Server` (إدارة الخادم) - للتحكم الكامل
    -   `Kick Members` (طرد الأعضاء)
    -   `Ban Members` (حظر الأعضاء)
    -   وغيرها حسب الحاجة.
4.  سيظهر لك رابط في الأسفل. انسخ هذا الرابط والصقه في متصفحك.
5.  اختر الخادم الذي تريد دعوة البوت إليه، ثم اضغط على `Authorize` (تفويض).

### 7. تشغيل البوت

1.  افتح ملف `bot.py`.
2.  ابحث عن السطر التالي:
    ```python
    # bot.run('YOUR_BOT_TOKEN')
    ```
3.  قم بإزالة علامة التعليق (`#`) من بداية السطر، واستبدل `'YOUR_BOT_TOKEN'` بالتوكن الذي نسخته من بوابة مطوري الديسكورد:
    ```python
    bot.run('YOUR_BOT_TOKEN')
    ```
    *نصيحة:* لتجنب وضع التوكن مباشرة في الكود، يمكنك تحميله من متغير بيئة. مثال:
    ```python
    import os
    from dotenv import load_dotenv

    load_dotenv() # حمل المتغيرات من ملف .env
    TOKEN = os.getenv('DISCORD_TOKEN')
    bot.run(TOKEN)
    ```
    إذا استخدمت هذه الطريقة، قم بإنشاء ملف جديد باسم `.env` في نفس مجلد `bot.py` وضع فيه سطرًا كهذا:
    ```
    DISCORD_TOKEN=YOUR_BOT_TOKEN_HERE
    ```
    وتذكر إضافة `python-dotenv` إلى `requirements.txt`.

4.  احفظ الملف `bot.py`.
5.  افتح سطر الأوامر وانتقل إلى مجلد المشروع، ثم قم بتشغيل البوت:
    ```bash
    python bot.py
    ```

الآن يجب أن يكون البوت قيد التشغيل في خادمك!

## الأوامر المتوفرة

-   `!count_members`: يعرض عدد الأعضاء في الخادم.
-   `!send_dm @اسم_العضو رسالتك هنا`: يرسل رسالة خاصة إلى العضو المحدد. (يتطلب صلاحيات إدارية لإطلاق هذا الأمر).

## إضافة المزيد من أوامر التحكم بالخادم

لإضافة أوامر تحكم مثل طرد الأعضاء، حظرهم، تغيير أسمائهم المستعارة، وما إلى ذلك، يمكنك إضافة المزيد من الدوال في `bot.py` باستخدام مكتبة `discord.py` وتعيين الصلاحيات المناسبة لها. مثال على أمر طرد عضو:

```python
@bot.command(name='kick')
@commands.has_permissions(kick_members=True)
async def kick(ctx, member: discord.Member, *, reason=None):
    """يطرد عضوًا من السيرفر."""
    await member.kick(reason=reason)
    await ctx.send(f'{member.display_name} تم طرده.')
```
